# AutoMarket Global

AutoMarket Global is a static demo website for a global car marketplace.

## Features

- Country selector
- Automatic language change based on selected country
- Arabic RTL support
- Search cars by country
- Filter by brand, fuel, price, year, and mileage
- Sell car demo form
- Car details page
- Favorites page
- Messages page
- Account page
- Dashboard page
- Terms page
- Demo data saved in browser localStorage
